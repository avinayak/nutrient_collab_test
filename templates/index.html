Nutrient Collaborative Test

This app demonstrates a collaborative signing flow using Nutrient's Document Engine.

Step 1: Upload & Prepare

Step 2: Generate Signing Links

Current Document ID: None




Instructions
1. Click "Generate Document" to create a new document with signature fields. (A predefined PDF with signature fields is used.)
2. After the document is created, use the "Generate Buyer Link" and "Generate Seller Link" buttons to get unique signing links for each role.


How It works:

  • The app uploads a static PDF with static signature fields and Instant JSON to the Nutrient Document Engine.
  • It fetches current annotations and creates a mapping between signature field widgets and BUYER/SELLER roles.
  • Assigns roles to signature fields based on this mapping. (PUT api/documents/{doc_id}/form-fields API)
  • It then generates unique JWT-authenticated links for "buyer" and "seller" roles.
  • Each link opens the document in a web viewer where the respective user can sign.
